# v1.2.0 Research Workflow Upgrade / 研究流程升级

This source release adds a small-budget research workflow for robustness analysis. It does not publish new model weights or replace the current default inference checkpoint until the Kaggle research run is completed and reviewed.

本源码版本新增小预算研究增强流程，用于稳健性分析。在 Kaggle 研究任务完成并审查前，不发布新的模型权重，也不替换当前默认推理 checkpoint。

## Highlights / 主要变更

- 3-fold cross-validation helpers with strict image/mask stem matching and leakage checks.
- Encoder comparison runner for high-capacity models such as U-Net++ with EfficientNet-B3 and ResNet34 encoders.
- Subgroup analysis by lesion size and image contrast, using per-sample segmentation metrics.
- Statistical summary utilities with mean, sample standard deviation, and 95% confidence intervals.
- Kaggle research script `notebooks/kaggle_research_v1_2.py` for end-to-end execution.
- Research artifact packaging support for `research_v1_2/**` outputs.
- Unit tests for cross-validation, subgroup analysis, and statistical analysis.

## Kaggle Command / Kaggle 命令

```bash
python notebooks/kaggle_research_v1_2.py
```

The script performs:

脚本执行：

1. Dependency and CUDA preparation.
2. `pytest`.
3. ISIC 2017 / ISIC 2018 data preparation.
4. Dataset sanity check.
5. 3-fold cross-validation on train+validation data.
6. Encoder comparison on the original train/validation split.
7. Threshold search for the selected encoder-comparison checkpoint.
8. Subgroup analysis on internal test and external validation splits.
9. Statistical summary generation.
10. Research artifact packaging.
