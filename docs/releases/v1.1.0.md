# v1.1.0 Reproducibility, Post-hoc Analysis, and Safety Upgrade / 可复现性、后处理分析与安全升级

This source release improves engineering reliability, adds post-hoc evaluation artifacts, and does not change the model weights published in v1.0.0.

本源码版本提升工程可靠性，补充后处理评估产物，不修改 v1.0.0 已发布的模型权重。

## Highlights / 主要变更

- Deterministic training controls and seeded DataLoader/Albumentations workers.
- Versioned checkpoints, `weights_only=True` loading, and architecture validation.
- Best-checkpoint prediction samples and unambiguous experiment-result fields.
- Specificity and Boundary F1 metrics plus optional independent test-split evaluation.
- Checkpoint-aware model caching and automatic architecture selection in Gradio.
- Pinned dependencies, GitHub Actions CI, pre-commit, security policy, dataset card, model card, and model manifest.
- Repeated-seed evaluation workflow for validation, independent ISIC 2017 test, and external ISIC 2018 validation.
- CPU/CUDA inference benchmark reporting.
- Threshold search, failure-case ranking, and visual failure-case exports.
- Release artifact package for experiment reports and representative visualizations.
- Expanded unit and smoke tests.

## Model Artifact / 模型权重

The verified U-Net++ EfficientNet-B3 checkpoint remains attached to [v1.0.0](https://github.com/dongtingshuo/medical-image-segmentation/releases/tag/v1.0.0). Its SHA256 and validation scope are recorded in `models/model_manifest.yaml` and `MODEL_CARD.md`.

已验证的 U-Net++ EfficientNet-B3 checkpoint 仍保存在 [v1.0.0](https://github.com/dongtingshuo/medical-image-segmentation/releases/tag/v1.0.0)。SHA256 和验证范围记录在 `models/model_manifest.yaml` 和 `MODEL_CARD.md`。

The recommended inference threshold is `0.35`, selected by validation Dice grid search from `0.30` to `0.70`.

推荐推理阈值为 `0.35`，该阈值由验证集 Dice 在 `0.30` 到 `0.70` 的网格搜索中选出。
